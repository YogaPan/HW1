如何編譯執行這份程式碼
======================

這份程式碼是用 C 語言寫的。
如果是在 Mac (Unix)、Linux 的環境下，直接在 Commnad Line 當前目錄下打 “make” 這個指令就會自動把編譯好的程式放到 “/bin” 資料夾下。
如果是在Windows作業系統下的話，在 “/bin” 裡就有已經先編譯好的exe檔，直接執行即可。


關於這份程式碼
==============

原始碼都放在 “/src” 資料夾裡，執行檔則在 "/bin" 資料夾下。
共有main_1.c, main_2.c, main_3.c三個程式，分別對應第一題、第二題還有第三題。
第一題的測資(3)和第二題沒有實作出來，其他部分都有寫出來。
